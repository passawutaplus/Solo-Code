# So1o Freelancer — UX Research Demo Guide

> **เช็คลิสครบ A–T:** [`ux-research-review.md`](./ux-research-review.md) · **ในแอป:** `/research`

> สำหรับทีม UX/UI Research ทดลองใช้งานเวอร์ชันเดโม่บน Vercel

---

## สรุปสั้นๆ

| คำถาม | คำตอบ |
|--------|--------|
| **ล็อกอิน / เก็บข้อมูล** | ได้จริง — บันทึกถาวร แยกตาม user (Google หรืออีเมล) |
| **ชำระเงิน** | sandbox เท่านั้น — ไม่มีการจ่ายจริง |
| **เข้าใช้** | สมัครได้เลย ไม่ต้องขออนุมัติ |

---

## วิธีเริ่มต้น (5 นาที)

1. เปิดลิงก์เดโม่ที่ทีมส่งให้ (Vercel Preview URL)
2. กด **เข้าสู่ระบบ** หรือ **สมัครสมาชิก**
3. แนะนำใช้ **Google Login** — เข้าได้ทันที ไม่ต้องยืนยันอีเมล
4. ถ้าใช้อีเมล+รหัส — ต้องกดลิงก์ยืนยันใน inbox ก่อนเข้า Dashboard
5. หลังล็อกอิน → ไปที่ **My Desk** (`/dashboard`)

## แบบฟอร์ม Feedback (กรอกหลังทดลอง)

สำหรับแต่ละ Task ที่ทำ ให้ให้คะแนน 1–5 และ comment สั้นๆ:

```
Task: [เช่น A4 สร้างใบเสนอราคา]
คะแนนความง่าย (1=ยากมาก, 5=ง่ายมาก): ___
คะแนนความสวย / น่าเชื่อถือ (1–5): ___
ติดขัดตรงไหน: ___
ข้อเสนอปรับปรุง: ___
```

### หัวข้อใหญ่ที่อยากได้ความเห็น

1. **First impression** — ภาพรวมแบรนด์ / ความน่าเชื่อถือ
2. **Navigation** — หาเมนูและฟีเจอร์ได้ไหม
3. **ภาษาไทย** — ข้อความเข้าใจง่ายไหม น้ำเสียงเหมาะกับฟรีแลนซ์ไหม
4. **Mobile UX** — ปุ่มกดง่ายไหม เนื้อหาอ่านได้ไหม
5. **AI features** — Assistant / AI ในหน้าต่างๆ ช่วยหรือรบกวน
6. **Trust** — กล้าใช้เก็บข้อมูลลูกค้า/การเงินจริงไหม

## ข้อมูลทดสอบตัวอย่าง (สร้างเองได้)

ใช้ข้อมูลสมมติได้เลย — ไม่ต้องเป็นข้อมูลลูกค้าจริง

| ฟิลด์ | ตัวอย่าง |
|--------|----------|
| ชื่อลูกค้า | คุณสมชาย ใจดี |
| อีเมล | test@example.com |
| โปรเจกต์ | ออกแบบโลโก้ร้านกาแฟ |
| ราคา | 15,000 บาท |
| กำหนดส่ง | +14 วัน |

## ความเป็นส่วนตัว

- บัญชีทดสอบบันทึกถาวร — อย่าใส่ข้อมูลลูกค้าจริงที่อ่อนไหว
- ข้อมูลแยกตาม user — คนอื่นมองไม่เห็น

## ติดปัญหา?

| อาการ | แนวทาง |
|--------|--------|
| Google login กลับมาแล้ว error | แจ้งทีม dev — ต้องเพิ่ม Vercel URL ใน Supabase redirect |
| อีเมลสมัครแล้วเข้าไม่ได้ | เช็ค inbox ยืนยันอีเมล หรือใช้ Google แทน |
| หน้าขาว / 500 | แคปหน้าจอ + URL ส่งทีม dev |
| AI ตอบไม่ได้ | เครดิต AI หมด — แจ้งทีม dev เติมเครดิตทดสอบ |
